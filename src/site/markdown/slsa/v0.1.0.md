# SLSA Provenance Build Type

This is a [SLSA Build Provenance](https://slsa.dev/spec/v1.2/build-provenance) build type to describe releases produced
by Apache Commons PMC release managers on their own equipment.

## Build definition

Artifacts are generated by a single Maven execution, usually of the form.

```shell
mvn -Prelease deploy
```
